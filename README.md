# Serverless Notes - 个人随笔系统

基于 Next.js + Supabase + Vercel 的 Serverless 架构个人随笔系统。

## 项目概述

这是一个完全免费的在线随笔管理系统，支持：
- 🌐 在线编辑：浏览器中直接写作和管理
- 📱 跨平台访问：PC + 手机响应式设计
- 🗄️ 云数据库：PostgreSQL 数据安全存储
- 🔐 用户认证：邮箱/GitHub 登录
- 📝 Markdown 编辑器：实时预览
- 🏷️ 标签系统：分类和搜索
- 📊 三级日志系统：完整追溯历史

## 技术栈

- **前端框架**: Next.js 14+ (App Router)
- **UI 组件**: Tailwind CSS + Shadcn/ui
- **数据库**: Supabase (PostgreSQL)
- **部署平台**: Vercel
- **编辑器**: Milkdown / ByteMD

## 模块

- 🎵 音乐 - 音乐相关的随笔和评论
- 🎬 电影 - 电影观后感和影评
- 📚 书籍 - 读书笔记和书评
- ⚽ 运动 - 运动记录和健身日志
- ✈️ 旅游 - 旅行游记和攻略

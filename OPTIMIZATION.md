## 最佳实践

### 1. 组件开发
- 使用 React.memo 避免不必要的重渲染
- 合理使用 useMemo 和 useCallback
- 避免在渲染函数中创建新对象/数组
- 使用 Suspense 和 lazy 进行代码分割

### 2. 数据获取
- 使用 SWR 或 React Query 进行数据缓存
- 实现乐观更新
- 合理使用 Supabase 的实时订阅
- 避免过度请求

### 3. 样式优化
- 使用 Tailwind CSS 的 JIT 模式
- 避免内联样式
- 合理使用 CSS 变量
- 优化动画性能（使用 transform 和 opacity）

### 4. 图片优化
- 使用 Next.js Image 组件
- 提供合适的图片尺寸
- 使用现代图片格式（WebP、AVIF）
- 实现懒加载

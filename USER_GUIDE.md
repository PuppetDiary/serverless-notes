## 快速开始

### 首次访问

1. 访问网站首页
2. 点击"登录"或"注册"按钮
3. 使用邮箱注册新账号
4. 查收邮箱验证邮件（如需要）
5. 登录后进入仪表盘

### 仪表盘概览

仪表盘是您的工作中心，包含：
- **统计卡片**：显示总随笔数、活跃模块数、最近变更数
- **模块概览**：五个领域的快速入口（音乐、电影、书籍、运动、旅游）
- **最近变更**：最新的操作日志

## 用户认证

### 注册账号

1. 点击"注册"按钮
2. 输入邮箱地址
3. 设置密码（至少6位）
4. 点击"注册"
5. 查收验证邮件（部分配置需要）

### 登录

1. 输入注册时使用的邮箱
2. 输入密码
3. 点击"登录"

### 登出

点击右上角用户菜单 → "登出"

## 创建随笔

### 方式一：从仪表盘创建

1. 点击右上角"写随笔"按钮
2. 选择所属模块（音乐/电影/书籍/运动/旅游）
3. 输入标题
4. 使用 Markdown 语法编写内容
5. 添加标签（可选）
6. 点击"发布"或"保存草稿"

### 方式二：从模块页面创建

1. 点击侧边栏中的模块（如"音乐"）
2. 点击"写随笔"按钮
3. 系统会自动选择该模块
4. 编写内容并发布

### Markdown 语法快速参考

```markdown
# 一级标题
## 二级标题
### 三级标题

**加粗文本**
*斜体文本*
~~删除线~~

- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

> 引用文本

[链接文本](https://example.com)

`行内代码`

​```
代码块
​```
```

### 实时预览

- 点击"预览"按钮查看渲染效果
- 点击"编辑"返回编辑模式
- 支持边写边预览

## 管理随笔

### 查看随笔

1. 从仪表盘或模块页面点击随笔卡片
2. 查看完整内容
3. 可以看到标题、内容、标签、创建时间等信息

### 编辑随笔

1. 打开随笔详情页
2. 点击"编辑"按钮
3. 修改内容
4. 点击"保存"或"发布"

### 删除随笔

1. 打开随笔详情页
2. 点击"删除"按钮
3. 确认删除操作
4. 随笔将被永久删除

### 随笔状态

- **草稿**：未发布的随笔，仅自己可见
- **已发布**：正式发布的随笔
- **已归档**：归档的随笔（未来功能）

## 使用标签

### 创建标签

1. 进入"标签管理"页面
2. 点击"新建标签"
3. 输入标签名称
4. 选择标签颜色
5. 保存

### 为随笔添加标签

在创建或编辑随笔时：
1. 找到"标签"选择器
2. 点击选择已有标签
3. 或创建新标签
4. 可以添加多个标签

### 按标签筛选

1. 进入"标签管理"页面
2. 点击某个标签
3. 查看该标签下的所有随笔

## 搜索功能

### 全文搜索

1. 点击侧边栏"搜索"
2. 输入关键词
3. 系统会搜索标题和内容
4. 点击结果查看详情

### 高级筛选

- 按模块筛选
- 按标签筛选
- 按日期范围筛选
- 按状态筛选（草稿/已发布）

## 查看日志

### 一级日志：系统概览

1. 点击侧边栏"系统日志"
2. 查看整体统计信息
3. 查看最近的操作记录
4. 点击模块卡片进入二级日志

### 二级日志：模块状态

1. 从一级日志点击某个模块
2. 查看该模块的统计信息
3. 查看该模块的随笔列表
4. 点击"详细日志"进入三级日志

### 三级日志：详细记录

1. 查看按日期分组的操作记录
2. 每条记录包含：
   - 操作类型（创建/更新/删除）
   - 操作描述
   - 操作时间
   - 相关随笔链接

### 日志导出

1. 在日志页面点击"导出"
2. 选择导出格式（Markdown）
3. 下载日志文件
4. 可用于备份或分析

## 常见问题

### Q: 忘记密码怎么办？

A: 目前需要联系管理员重置。未来版本将支持密码重置功能。

### Q: 可以上传图片吗？

A: 图片上传功能正在开发中，即将推出。

### Q: 数据会丢失吗？

A: 所有数据存储在 Supabase 云数据库中，安全可靠。建议定期导出备份。

### Q: 支持多人协作吗？

A: 当前版本为个人使用设计。多人协作功能在规划中。

### Q: 可以导出数据吗？

A: 支持导出日志为 Markdown 格式。完整数据导出功能即将推出。

### Q: 支持离线使用吗？

A: 离线功能（PWA）正在开发中。

### Q: 如何备份数据？

A: 
1. 定期导出日志
2. 使用 Supabase Dashboard 导出数据库
3. 未来将支持一键备份功能

### Q: 遇到错误怎么办？

A: 
1. 尝试刷新页面
2. 清除浏览器缓存
3. 检查网络连接
4. 查看错误详情（开发环境）
5. 联系技术支持
